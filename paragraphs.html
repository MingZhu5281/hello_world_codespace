GitHub Codespaces free tier (for personal accounts) includes:

Compute hours: 120 core hours per month (equivalent to 60 hours on a 2-core machine or 30 hours on a 4-core machine). Storage: 15 GB per month. Billing: Usage beyond these limits incurs charges based on compute time and storage. For the latest details, check the GitHub Billing documentation. Note that limits may vary by account type (e.g., organizations have different tiers).
